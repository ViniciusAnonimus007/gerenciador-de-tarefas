# gerenciador-de-tarefas

O projeto de gerenciador de tarefas tem como objetivo testar meus conhecimentos básicos em CRUD. Futuramente pode ser evoluído para algo utilizável

O projeto consiste em uma página onde pode se ver algumas informações como: 
- Tarefas pendentes
- Criar novas tarefas
- Deletar/resolver tarefas
